# Глоссарий

Список терминов, используемых в переводе. Если термин встречается впервые — добавьте его сюда.

| Английский термин            | Русский перевод / транслитерация | Примечание                                           |
| ---------------------------- | -------------------------------- | ---------------------------------------------------- |
| Agent                        | Агент                            |                                                      |
| Agentic                      | Агентный                         | Связанный с автономными агентами                     |
| Agentic Engeneerig           | Агентная Инженерия               | Новый подход в разработке систем                     |
| Agentic Engineering Patterns | Паттерны Агентной Инженерии      | Шаблоны для Агентной Инженерии                       |
| Coding agent                 | Агент для программирования       | Инструмент, который генерирует и выполняет код       |
| Context window               | Контекстное окно                 |                                                      |
| Guide                        | Гайд                             | Формат материала, объединяющий несколько глав        |
| Orchestration                | Оркестрация                      |                                                      |
| Pattern                      | Паттерн / Шаблон                 | «Паттерн» предпочтительно в контексте проектирования |
| Pipeline                     | Пайплайн                         |                                                      |
| Prompt                       | Промпт                           | Устоявшееся заимствование                            |
| Reasoning                    | Рассуждение                      |                                                      |
| Tool use                     | Использование инструментов       |                                                      |
| Vibe coding                  | Вайб кодинг                      | Обычно оставляется как термин без полного перевода   |
| Workflow                     | Рабочий процесс / Воркфлоу       |                                                      |

> Термины добавляются по алфавиту (по английскому названию).

# Repository Guidelines

This repository contains a Russian translation of Simon Willison's "Agentic Engineering Patterns".

## Language Quality

When reviewing Russian text, also check for:

- spelling mistakes and typos
- punctuation errors
- awkward or unnatural phrasing in Russian
- inconsistent capitalization of terms and headings
  Only comment on language issues when they are clear and actionable.
  Do not suggest purely subjective rewrites if the current text is already correct and readable.
